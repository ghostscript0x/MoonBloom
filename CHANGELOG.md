# Changelog

All notable changes to Moon Bloom Tracker will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release of Moon Bloom Tracker PWA
- AI-powered cycle prediction and insights
- Comprehensive period, mood, and symptom tracking
- Wellness integration (exercise, sleep, nutrition)
- Native device notifications
- App lock with biometric/PIN authentication
- Data export functionality
- Offline-first architecture
- PWA installation support

### Features
- Cycle length editing (21-45 days)
- Real-time cycle phase detection
- Health insights and trend analysis
- Calendar view with cycle visualization
- Quick logging with floating action button
- Privacy-focused data management
- Cross-platform compatibility

## [1.0.0] - 2025-01-01

### Added
- Complete menstrual cycle tracking system
- User authentication and profile management
- Local data storage with encryption
- Responsive design for mobile and desktop
- Accessibility features (WCAG 2.1 AA compliant)
- Comprehensive privacy policy and terms of service

### Technical
- React 18 with TypeScript
- Tailwind CSS + Shadcn/ui component system
- Progressive Web App with service workers
- IndexedDB for local storage
- Chart.js integration for data visualization
- Biometric authentication support

### Security
- End-to-end data encryption
- Secure authentication flows
- Privacy-first data handling
- No third-party data sharing

- **MINOR** version for backwards-compatible functionality additions
- **PATCH** version for backwards-compatible bug fixes
