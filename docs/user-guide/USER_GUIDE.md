### Understanding Your Cycle

Moon Bloom Tracker uses a standard menstrual cycle model:

- **Menstrual Phase** (Days 1-5): Period flow
- **Follicular Phase** (Days 6-14): Hormone rise, energy increase
- **Ovulation Phase** (Day 14-16): Fertile window
- **Luteal Phase** (Days 15-28): PMS symptoms, period preparation
